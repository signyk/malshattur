# Málsháttur

A minimal website that gives you a random Icelandic proverb (*málsháttur*) at the click of a button.

**Live at [malshattur.is](https://malshattur.is)**

## What it does

- Fetches the full list of proverbs from the [Icelandic Wikipedia](https://is.wikipedia.org/wiki/Listi_yfir_%C3%ADslenska_m%C3%A1lsh%C3%A6tti) on page load
- Displays a random proverb each time you click "Fá málshátt"
- Lets you share the proverb via the native share sheet (mobile) or clipboard (desktop)

## Stack

Plain HTML, CSS, and JavaScript — no framework, no build step. Hosted on Cloudflare Pages with automatic deploys on push to `main`.

## Development

Just open `index.html` in a browser. No dependencies to install.
